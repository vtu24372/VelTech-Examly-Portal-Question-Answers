Write a Java method that performs the run length encoding for a given String and returns the run length encoded String.


You just need to count the number of times a character is occurring and concat it after the char. Repeat the process for all unique characters in the given string and return the final answer.


Sample input: ABBBB


Sample output: A1B4


Explanation:  Character A occurs 1 time, and character B occurs 4 times. Hence, the answer is printed as A1B4.  
Input format :

One string is given as input
Output format :

One string is printed on the screen
Sample test cases :
Input 1 :

ABBCCC

Output 1 :

A1B2C3

Note :
The program will be evaluated only after the “Submit Code” is clicked.
Extra spaces and new line characters in the program output will result in the failure of the test case. 

import java.util.*;
import java.util.Scanner;

class RunLengthEncoding{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String s =sc.next();
        String result ="";
        int count =1;
        
        for(int i=0; i<s.length(); i++){
            
            if(i<s.length()-1 && s.charAt(i)==s.charAt(i+1))
            count++;
            
            else{
                result += s.charAt(i)+""+count;
                count =1;
            }
        }
        
        System.out.println(result);
    }
}

 Single File Programming Question

Write a Java method that performs the encoding for a given String as explained below and returns a factorial of the maximum count.


You just need to count the number of times a character is occurring and find the maximum count among them. Find the factorial for that number and return the answer.


Sample input: ABBBB


Sample output: 24


Explanation:  Character A occurs 1 time, and character B occurs 4 times. The maximum value is 4 and 4! = 24. Hence, 24 is printed.  


Note on finding the factorial: 


 	Factorial of N = N * (N-1) * (N-2) * ... * 1 

Factorial of 1 = 1 

 	Factorial of 4 = 4 * 3 * 2 * 1 = 24 

 	Factorial of 5 = 5 * 4 * 3 * 2 * 1 = 120 
Input format :

One string is given as input
Output format :

One integer is printed on the screen
Sample test cases :
Input 1 :

ABBCCC

Output 1 :

6

Note :
The program will be evaluated only after the “Submit Code” is clicked.
Extra spaces and new line characters in the program output will result in the failure of the test case. 

import java.util.*;
import java.util.Scanner;

class FactorialOfMaxCount{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String s=sc.next();
        int max=0;
        
        for(char c:s.toCharArray()){
            int count =0;
            
            for(char d:s.toCharArray()){
                if(c==d) count++;
                if(count>max) max=count;
            }
        }
        
        int fact=1;
        for(int i=1; i<=max; i++){
            fact*=i;
        }
        System.out.print(fact);
    }
}

 Single File Programming Question

Care hospital wants to know the medical specialty visited by the maximum number of patients. The details of the medical specialties are as follows: {“P":"Pediatrics", "O":"Orthopedics", "E":"ENT}


Assume that the patient id of the patient along with the medical specialty visited by the patient is stored in a String variable. 

Write a Java method to find the medical specialty visited by the maximum number of patients and return the name of the specialty. 


Note 1: Assume that there is always only one medical specialty that is visited by a maximum number of patients.

Note 2: Perform case-sensitive string comparison wherever necessary.

Note 3:   input is a String where the values are separated by a comma.


Sample input: 

 list=“101,O,102,E,302,P,305,P,401,E,656,O,987,E"


Sample output: 

           ENT 


Explanation: 

In the given list, more patients visited the E department which means the ENT department. Hence, ENT is printed as the answer.
Input format :

One string is given as input
Output format :

One String is printed on the screen
Sample test cases :
Input 1 :

1,E,1,O,1,P,2,P

Output 1 :

Padiatrics

Note :
The program will be evaluated only after the “Submit Code” is clicked.
Extra spaces and new line characters in the program output will result in the failure of the test case.

import java.util.*;
import java.util.Scanner;

class PatientMaxCount{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String s = sc.nextLine();
        String[] a = s.split(",");
        
        int p=0,o=0,e=0;
        
        for(int i=1; i<a.length; i+=2){
            if(a[i].equals("P")) p++;
            else if(a[i].equals("O")) o++;
            else if(a[i].equals("E")) e++;
        }
        
        if(p>o && p>e) System.out.print("Padiatrics");
        
        else if(o>p && o>e) System.out.print("Orthopedics");
        
        else System.out.print("ENT");
    }
}

    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String s = sc.nextLine().replace("","");
        String[] a = s.split(",");
        
        Set<Character> set = new TreeSet<>();
        
        for(char c:a[0].toCharArray()){
            if(a[1].indexOf(c)!=-1)
            set.add(c);
        }
        
        if(s.isEmpty()) System.out.print("None");
        for(char c:set){
            System.out.print(c);
        }
    }

import java.util.*;

class CommonVowel{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String s = sc.nextLine().replace(" "," ");
        String[] a = s.split(",");
        
        String vowel = "aeiou";
        
        Set<Character> set = new TreeSet<>(); 
        
        for(char c: a[0].toCharArray()){
            if(vowel.indexOf(c)!=-1 && a[1].indexOf(c)!=-1)
            set.add(c);
        }
        
        if(set.isEmpty()) System.out.print("None");
        else for(char c:set) System.out.print(c);
    }
}
