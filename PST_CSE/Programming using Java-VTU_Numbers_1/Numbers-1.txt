 Single File Programming Question

Write a method to print sum of first N natural numbers.


The natural numbers are the numbers that include all the positive integers from 1 to infinity.


Sum of first n natural numbers, can be obtained using the following formula:

sum = (n*(n+1)) /2 


Example

Sample input : 4

Sample output: 10

Explanation: We need to add numbers from 1 to 4.

1 + 2 + 3 + 4 = 10.

So, return 10 as answer.
Input format :

An integer value is given as input.
Output format :

Print sum of first N natural numbers, for the given number N.
Sample test cases :
Input 1 :

4

Output 1 :

10

The program will be evaluated only after the “Submit Code” is clicked.

class NaturalNumbers{
    
    static int sumOfNumbers(int n){
    int sum=0;
for(int i=0; i<n; i++){
    sum=(n*(n+1))/2;
}
return sum;


}
public static void main(String s[]){
    int n = new java.util.Scanner(System.in).nextInt();
    System.out.println(sumOfNumbers(n));
    }
}

 Single File Programming Question

(repeated) You are given a number.

The task is to find the factorial of the given number using recursive method.


Example:


Factorial of a number n is defined as: 


n! = n * (n-1) * (n-2) * .... 1 


0 ! = 1

1 ! = 1 

3 ! = 3 * 2 * 1 = 6. 

4 ! = 4 * 3 * 2 * 1 = 24. 

5 ! = 5 * 4 * 3 * 2 * 1 = 120. 
Input format :

An integer value is given as input.
Output format :

Return the factorial of given number.
Sample test cases :
Input 1 :

5

Output 1 :

120

Note :
The program will be evaluated only after the “Submit Code” is clicked.
Extra spaces and new line characters in the program output will result in the failure of the test case. 

class FindFactorial{
    
    static int factorial(int n){
    if(n==0 || n==1){
    return 1;
} else{
    return n*factorial(n-1);
}

}

public static void main(String s[]){
    int n = new java.util.Scanner(System.in).nextInt();
    System.out.println(factorial(n));
    }
}

 Single File Programming Question

Given an input n, find the sum of digits in the integer n and the factorial n.

Example:

Assume given number n = 123. First we need to find the sum of digits in the n, the sum of the digits in n are 1+2+3=6.

After finding the sum of digits in n, find the factorial of the sum.

Here, the factorial of 6 = 6*5*4*3*2*1 = 720.
Input format :

Input contains an integer denoting n
Output format :

Output displays the factorial of the sum of the integer n.
Sample test cases :
Input 1 :

123

Output 1 :

720

Note :
The program will be evaluated only after the “Submit Code” is clicked.
Extra spaces and new line characters in the program output will result in the failure of the test case. 

class SumDigitsFactorial{
    
    static int fact(int number){
        int sum = 0;
        
        while(number > 0){
            sum = sum + number % 10;
            number = number / 10;
        }

        int f = 1;
        for(int i = 1; i <= sum; i++){
            f = f * i;
        }
        
        return f;
    }

    public static void main(String s[]){
        int n = new java.util.Scanner(System.in).nextInt();
        System.out.print(fact(n));
    }
}
