 Single File Programming Question

Write a program to implement the Stack using an array. Given a stack consisting of ‘n’ elements, perform ‘m’ number of pop operations and print the resulting stack. 


Note: While printing the stack, the elements have to be separated by a space and nothing is printed after the last element. Assume, that m < n.
Input format :

An integer 'n', followed by 'n' integers representing 'n' values of the array, and another interger 'm' are given as input.
Output format :

Print the resulting stack on the screen.
Sample test cases :
Input 1 :

5
1 2 3 4 5
3

Output 1 :

1 2 

Input 2 :

8
8 7 6 5 4 3 2 1
6

Output 2 :

8 7 

Note :
The program will be evaluated only after the “Submit Code” is clicked.
Extra spaces and new line characters in the program output will result in the failure of the test case. 

import java.util.*;

class PopUsingStack{
    public static void main(String[] args){
        Scanner s = new Scanner(System.in);
        
        int n = s.nextInt();
        int[] a = new int[n];
        
        for(int i=0; i<n; i++)
        a[i] = s.nextInt();
        
        int m = s.nextInt();
        
        for(int i=0; i<n-m; i++){
            if(i==n-m-1)
            System.out.print(a[i]);
            else
            System.out.print(a[i] + " ");
        }
    }
}
