 Single File Programming Question

Write a program to delete an 'element' from the given position of the linked list. You need to read the 'size’ and the 'elements' of the linked list. Also, read the 'position', from where the element has to be deleted. 

Assume that, the element to be deleted always exists in the linked list.
Input format :

Line 1: Number of elements in the linked list

Line 2: Elements of linked list

Line 3: The position from where the element to be deleted.
Output format :

Print the resulting latest linked list elements in the same line. The elements have to be separated by a space and nothing is printed after the last element.
Sample test cases :
Input 1 :

5
10 20 30 40 50
2

Output 1 :

10 30 40 50 

Input 2 :

3
100 200 300
1

Output 2 :

200 300 

Note :
The program will be evaluated only after the “Submit Code” is clicked.
Extra spaces and new line characters in the program output will result in the failure of the test case. 

import java.util.*;

class DeleteAtLinkedList{
    public static void main(String[] args){
        Scanner s = new Scanner(System.in);
        
        int n = s.nextInt();
        int[] a = new int[n];
        
        for(int i=0; i<n; i++)
        a[i] = s.nextInt();
        
        int pos = s.nextInt();
        
        for(int i=0; i<n; i++){
            if(i != pos-1){
            System.out.print(a[i]);
            
            if(i != n-1 && !(i==n-1 && pos-1==n-1))
            System.out.print(" ");
            }
        }
    }
    
}

 Single File Programming Question

Write a program to insert an element in the given ‘position’ of the linked list. You need to read ‘n’ elements of the linked list. Also, read the new element and its position, where the new element has to be inserted. 
Input format :

Line 1: Number of elements in the linked list

Line 2: Elements of linked list

Line 3: The new element to be inserted

Line 4: The position, where the new element has to be inserted
Output format :

You have to print the latest linked list elements in the same line. The elements to be separated by a space and nothing is printed after the last element.
Sample test cases :
Input 1 :

4
1 2 3 5
4
4

Output 1 :

1 2 3 4 5 

Input 2 :

3
66 88 99
77
2 

Output 2 :

66 77 88 99 

Note :
The program will be evaluated only after the “Submit Code” is clicked.
Extra spaces and new line characters in the program output will result in the failure of the test case. 

import java.util.*;

class InsertAtLinkedList{
    public static void main(String[] args){
        Scanner s = new Scanner(System.in);
        
        int n = s.nextInt();
        int[] a = new int[n+1];
        
        for(int i=0; i<n; i++)
        a[i] = s.nextInt();
        
        int x = s.nextInt();
        int pos = s.nextInt();
        
        for(int i=n; i>pos-1; i--){
            a[i] = a[i-1];
        }
        a[pos-1] = x;
        
        for(int i=0; i<n+1; i++){
            if(i==n)
            System.out.print(a[i]);
            else
            System.out.print(a[i] + " ");
        }
    }
}


 Single File Programming Question

Write a program to reverse the elements in the linked list. You need to read the ‘size’ and 'elements' of the linked list. 
Input format :

Line 1: Size of the linked list.

Line 2: Elements of the linked list seperated by single space.
Output format :

Print the resulting linked list elements separated by a space.
Sample test cases :
Input 1 :

5
10 20 30 40 50

Output 1 :

50 40 30 20 10 

Input 2 :

6
6 5 4 3 2 1

Output 2 :

1 2 3 4 5 6 

Note :
The program will be evaluated only after the “Submit Code” is clicked.
Extra spaces and new line characters in the program output will result in the failure of the test case. 

import java.util.*;

class ReverseLinkedList{
    public static void main(String[] args){
        Scanner s = new Scanner(System.in);
        
        int n = s.nextInt();
        int[] a = new int[n];
        
        for(int i=0; i<n; i++)
            a[i] = s.nextInt();
            
        for(int i=n-1; i>=0; i--){
            if(i==n)
            System.out.print(a[i]);
            else
            System.out.print(a[i] + " ");
        }
    }
}
